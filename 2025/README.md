# 2025

## Day 1 – Awk

```console
$ awk -f day01-01.awk < input1.txt
$ awk -f day01-02.awk < input1.txt
```

## Day 2 – Python

```console
$ python3 day02-01.py < input2.txt
$ python3 day02-02.py < input2.txt
```

## Day 3 – Haskell

```console
$ runhaskell day03-01.hs < input3.txt
$ runhaskell day03-02.hs < input3.txt
```

## Day 4 – APL

With Dyalog APL installed, run:

```console
$ ./day04-01.apls
$ ./day04-02.apls
```

## Day 5 – Factor

```console
$ Get-Content 'input5.txt' | factor.com day05.factor
```

## Day 6 – (Chicken) Scheme

```
$ csi -s day06.scm < input6.txt
```

## Day 7 – Snobol

```
$ snobol4 day07-01.sno < input7.txt
$ snobol4 day07-02.sno < input7.txt
```
## Day 8 – Fortran

```
$ gfortran -O2 day08-01.f90 < input8.txt
$ gfortran -O2 day08-02.f90 < input8.txt
```
